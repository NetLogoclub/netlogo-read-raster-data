一般生产asc栅格数据有两个文件，一个是asc文件，一个是prj投影文件，把prj文件放到别的目录里先存着，先只用asc文件，这样比较方便，到时候导出asc文件，然后把prj文件和它放同一个目录里，就可以放arcmap中出图了

设置：瓦片大小设置应该和asc文件的ncols 和nrows大小一样。本demo设置瓦片世界大小1005*957，asc文件的ncols*nrows 也是1005*957。这样栅格数据和瓦片一一对应，根据瓦片里对应的栅格值可以着色！
